# VoiceToText - Оффлайн распознавание речи

Оффлайн-приложение для распознавания русской речи с графическим интерфейсом на основе библиотеки Vosk.

##  Особенности

- **Полностью оффлайн** - не требует интернет-соединения для работы
- **Русский язык** - модель распознавания уже включена в проект
- **Графический интерфейс** - удобный интерфейс на Java Swing
- **Работа в реальном времени** - отображение частичных результатов во время речи
- **Кроссплатформенность** - работает на Windows, macOS, Linux
- **Простота использования** - запуск в несколько кликов

##  Системные требования

- **Java 11** или выше
- **Maven 3.6** или выше
- **Микрофон** (встроенный или внешний)

## Используемые технологии
- Vosk 0.3.45 - оффлайн-библиотека распознавания речи
- Java Swing - графический интерфейс
- Java Sound API - работа с аудио-вводом
- Maven - управление зависимостями и сборка

##  Быстрый старт

### Клонирование и сборка:

```bash
git clone https://github.com/Hawkeye8270/VoiceToText.git

cd VoiceToText
mvn clean package
```


### Запуск приложения:
```
java -jar target/voice-to-text-1.0.0.jar 
```

### Сборка проекта
Компиляция:
```
mvn clean compile
```

### Создание исполняемого JAR-файла:
````
mvn clean package
````

После успешной сборки в папке target будет создан файл VoiceToText-1.0-SNAPSHOT.jar


### Альтернативные способы запуска:
Через Maven:

```bash
mvn exec:java -Dexec.mainClass="com.speech.recognition.SpeechToTextApp"
```

Из IDE:

* Импортируйте проект как Maven проект
* Запустите класс com.speech.recognition.SpeechToTextApp

## Использование приложения
- Запустите приложение одним из способов выше
- Дождитесь сообщения "Система готова к работе" в статусной строке
- Нажмите кнопку "Начать запись" (зеленая)
- Говорите в микрофон - текст будет появляться в текстовой области в реальном времени
- Нажмите "Остановить запись" (красная) для завершения сеанса
